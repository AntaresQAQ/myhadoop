# My Hadoop

## Environment

Linux amd64 / MacOS amd64

## Requirements
- docker
- wget

## Get Start

### build and start
```bash
git clone https://github.com/AntaresQAQ/myhadoop.git
cd myhadoop
chmod +x download.sh build-and-run.sh
./download.sh
./build-and-run.sh
```

### start compose
```bash
docker-compose start
```

### stop
```bash
docker-compose stop
```
